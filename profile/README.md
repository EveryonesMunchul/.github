## 🗳️ 모두의 문철이란?

내가 맞다 네가 맞다 끝없는 논쟁, **익명 투표로 판결받는 커뮤니티 재판소**입니다.
일상의 모든 빡침을 사연으로 올리면, 사람들이 둘 중 하나를 골라 투표로 판결합니다.

<br>

## ✨ 핵심 기능

<p align="center">
  <img src=".github/features.png" alt="핵심 기능 - 실시간 투표, 익명 가능, 둘 중 택1 판결" width="100%">

- **⚡ 실시간 투표** — 표가 뒤집히는 손맛. 마감 직전 역전극은 직접 봐야 압니다.
- **🕶️ 익명 가능** — 사연도 투표도 익명으로. 눈치 안 보고 솔직하게.
- **⚖️ 둘 중 택1 판결** — A냐 B냐 한쪽을 골라요. 마감되면 다수결로 판결 확정.
